How to Use

Quick reference for upload, transform, filtering, and channel adjustments.

Controls

  • Upload: Drag and drop an image into the workspace or click to browse.
  • Move: Drag the image or use arrow keys to reposition.
  • Zoom: Use +/- buttons, keyboard keys, or mouse wheel.
  • Flip: Press F or use the Flip control.
  • Rotate: Press R or use the Rotate control.
  • Clear: Resets the current session and clears the loaded image.

Filters

  • Undo / Redo: Step backward or forward through edit history.
  • Invert: Inverts image colors.
  • Grayscale: Converts to grayscale tones.
  • Dark Edges: Darkens toward edges for a vignette-like result.
  • Beige: Applies a warm beige tint.
  • Polarization: Reduces glare and boosts perceived saturation.
  • Acrylic: Applies a painterly acrylic-style look.
  • Median: Reduces salt-and-pepper style noise.
  • Sobel: Highlights edges and high-contrast boundaries.

Map Generation

  • Height: Builds a height map from the current image.
  • Normal: Builds a normal map from the current image.

Channels

  • Red / Green / Blue: Adjust individual color channel intensity.
  • Alpha: Adjust overall transparency.
  • Brightness: Adjust overall luminance.

Layers

  • New Layer: Creates a blank top layer above the current image.
  • Canvas Grab Tool: Drags the whole canvas view (default).
  • Layer Grab Tool: Drags only the selected layer in the layer list.
  • Duplicate Base: Creates a top layer from the current image.
  • Hide Top Layer: Toggles top layer visibility.
  • Mode: Controls blend mode for the selected layer.
  • Merge Down: Bakes the current blend into a single image.
  • Discard Layer: Removes the top layer and restores base image.
